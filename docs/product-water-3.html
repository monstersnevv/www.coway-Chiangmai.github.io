    สินค้าแนะนำ
    หมวดหมู่ : เครื่องกรองน้ำ Coway
    ยี่ห้อสินค้า : Coway
    รหัสสินค้า : P-300R (นาดี)

    เครื่องกรองโคเวย์ รุ่น P-300R (นาดี) น้ำธรรมดา

    ระบบการกรอง R.O จำเป็นต้องใช้ปั๊มดันน้ำในพื้นที่แรงดันต่ำ

    คุณสมบัติเครื่องกรองน้ำ Coway

    ถังเก็บน้ำสูญญากาศ : ระบบฝาปิดถังเก็บน้ำป้องกันฝุ่นละอองและแมลงเข้าสู่ถังเก็บน้ำ

    ควบคุมแรงดันน้ำ : ระบบควบคุมแรงดันน้ำ เพื่อป้องกันระบบการกรองภายใน ซึ่งอาจทำให้เครื่องกรองน้ำได้รับความเสียหายจากแรงดันน้ำสูง

    ระบบควบคุมน้ำล้น 2 ชั้น : เทคโนโลยี DFLC ควบคุมระดับน้ำอัตโนมัติ 2 ชั้น ป้องกันน้ำล้นในถังเก็บน้ำได้อย่างมั่นใจ

    ถังเก็บน้ำขนาดใหญ่ : ขนาดถังเก็บน้ำที่ใหญ่กว่า เมื่อเทียบกับเครื่องกรองน้ำขนาดเดียวกัน จึงสามารถกรองน้ำสะอาดได้อย่างเพียงพอต่อความต้องการ

    บูสเตอร์ปั๊ม : เพิ่มประสิทธิภาพการกรองด้วยบูสเตอร์ปั๊มเพิ่มแรงดันน้ำ ทำให้สามารถใช้งานได้ แม้ในแหล่งที่มีแรงดันน้ำต่ำ

    ระบบป้องกันแบคทีเรียเจริญเติบโต : ป้องกันการเกิดแบคทีเรียในถังเก็บน้ำอย่างมีประสิทธิภาพด้วยแอนตี้แบคทีเรียฟิลเตอร์

    - ผลิตน้ำพร้อมดื่มได้ 4 อุณหภูมิ น้ำร้อน + น้ำอุ่น , น้ำเย็น + น้ำปกติ
    - ถังเก็บน้ำขนาดใหญ่รวม 11.3 ลิตร
    - ออกแบบมาให้ใช้งานง่าย User friendly Design
    - ปุ่มล็อคน้ำร้อน เด็ก ๆ ใช้งานได้อย่างปลอดภัย
    - ประหยัดไฟด้วยเทคโนโลยีกักเก็บแบตเตอร์รี่ไฟฟ้าจากโคเวย์
    - ระบบประหยัดพลังงานอัตโนมัติ : ด้วยเซ็นเซอร์ตรวจวัดระดับแสง ปรับเข้าโหมดกลางคืนอัตโนมัติเมื่อห้องมืดหรือมีแสงน้อย
    - บูสเตอร์ปั๊ม สำหรับแหล่งน้ำที่มีแรงดันน้ำต่ำ
    - ถังเก็บน้ำขนาดใหญ่ : ด้วยถังน้ำขนาดใหญ่ที่มาพร้อมกับประสิทธิภาพการทำความเย็น ทำให้แน่ใจได้ว่าคุณจะมีน้ำเย็นเพียงพอต่อความต้องการ
    - ระบบป้องกันแบคทีเรียเจริญเติบโต : ป้องกันการเกิดแบคทีเรียในถังเก็บน้ำอย่างมีประสิทธิภาพด้วยแอนตี้แบคทีเรีย (Anti-Bacterial Filter)


    • ขนาดสินค้า (กว้าง x สูง x ลึก) : 31 × 40 × 40.5
    • น้ำหนัก 8 kg.
    • 13 W.


    คุณภาพไส้กรองที่มาตรฐานโลกรับรอง NSF/NASIไส้กรอง
    ทำหน้าที่ ดัก+จับ+แยก+ปรับ+ยับยั้ง

    ประกอบด้วย :

    - ไส้กรอง นีโอ เซ็นต์ NEO-SENSE FILTER (Plus Sediment + Carbon ) ทำหน้าที่ดัก+จับ ดักกรองเศษสนิม ดินทราย ตะกอนความขุ่นที่อยู่ในน้ำ
    - ไส้กรองพรีคาร์บอน (Pre-Carbon) ทำหน้าที่จับ จับกลิ่น สี คลอรีนต่าง ๆ ที่ปะปนอยู่ในน้ำ
    - ไส้กรองอาร์โอ (RO-Membrane) ทำหน้าที่แยก คัดแยกสารละลาย โลหะหนัก ต่างๆออกจากน้ำดี
    - ไส้กรองพลัสอินโนเซ้นท์ (Plus INNO-Sense) ทำหน้าที่ปรับ เป็นการกรองครั้งสุดท้ายและปรับรสชาติของน้ำ
    - แอนตี้แบคทีเรีย (Anti Bacterial) ทำหน้าที่ยับยั้ง ยับยั้งการเจริญเติบโตของจุลชีพในถังเก็บน้ำ
